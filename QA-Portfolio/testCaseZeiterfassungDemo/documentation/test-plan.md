# Testplan - ZeiterfassungDemo - App

## Projekt 
ZeiterfassungsDemo - App

## Ziel
Überprüfung der
wichtigsten Funktinonen vor Veröffentlichung

## Testfälle

-   Login
-   Zeiterfassung
-   Datenpüfung
-   Benutzeraktionen

## Testfälle

- TC-001 Login erfolgreich
- TC-002 Login mit falschen Passwörtern 
- TC-003 Pflichtfeldprüfung
- TC-004 Zeiteintrag erstellen
- TC-005 Zeiteintrag bearbeiten

## Fehlerverwaltung
Gefundene Fehler werden dokumentiert mit:
- Bug ID
- Beschreibung
- Schritte zum Nachstellen
- Erwartetes Ergebnis
- Tatsächliches Ergebnis
- Schweregrad

## Abschluss

Der Test ist abgeschlossen wenn:
-  alle geplnten Testflälk ausgeführt wurden
-  offene Fehler bewertet
