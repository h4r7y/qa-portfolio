##  ST002 - Navigation und Übersicht prüfen

**System:** <br>
Zeiterfassungs-Demo Anwendung

**Ziel:** <br>
Prüfen, ob die wichtigesten Bereiche erreichbar sind.

**Schritte:** <br>
1.  Anwendung öffnen
2.  Benutzer anmelden
3.  Dashboard öffnen
4.  Zur Zeiterfassung navigieren
5.  Übersicht der Einträge öffnen

**Erwartetes Ergebnis:** <br>
-  Anwendung lädt ohne Fehler
-  Dashboard wird angezeigt
-  Zeiterfassung ist erreichbar
-  Vorhandene Einträge werden angezeigt

**Ergebnis:** <br>
Bestanden
