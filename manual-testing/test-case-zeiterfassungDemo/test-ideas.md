# QA Test Ideas

## 001 - Mehrfachklick auf Sprichern

### Szenario
Der Benutzer klickt mehrfach auf "Speichern", weil keine Rückmeldung erscheint.

### Erwartung
- Nur Datensatz wird gespeichert
- Button wird deaktiviert
- Ladesnimation erscheint
- Keine Fehlermeldung

## 002 - Galerie

### Szenario 
100x auf "Weiter" klicken

### Erwartung
- Kein Absturz
- Alle Bilder werden geladen
- Richtiger Bildzähler
